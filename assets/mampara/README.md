# Assets mampara (sanitizados)

Estos archivos se extrajeron del PPTX de referencia para replicar el layout de la "mampara"
en `pptxgenjs` **sin** incluir fotografia(s) ni textos de un caso real.

Incluye:
- `header-bar.png`: barra superior
- `left-stripes.png`: fondo de panel izquierdo
- `pattern.png`: patron de fondo (columna derecha)
- `logo-fge-puebla.png`: logotipo
- `slogan.png`: leyenda inferior derecha

No incluye la fotografia embebida del PPTX (contenido sensible).
