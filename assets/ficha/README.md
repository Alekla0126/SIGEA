# Assets (Ficha PPTX)

Estos recursos son **graficos de referencia** (logos/escudos) usados para generar la **Ficha (PPTX)** con el layout del ejemplo.

- `logo-fiscalia.png`: logo horizontal (esquina superior derecha)
- `escudo.png`: recorte del emblema (esquina superior izquierda)

Nota: No se incluyen plantillas `.pptx` de referencia ni contenido sensible (texto/fotos reales de casos).
